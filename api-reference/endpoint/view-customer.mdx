---
title: 'View Recipient'
openapi: 'GET /customer/{customer_id}'
---

### Description

This endpoint retrieves the details of a specific recipient (customer) in your business account. The recipient is identified by a unique ID and includes their name, email, phone number.
The endpoint also returns any transactions associated with the recipient.

### Request

```http
GET /customer/{customer_id}
```

### Headers

```http
api-key: your_secret_key_here
```
